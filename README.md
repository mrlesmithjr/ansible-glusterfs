# ansible-glusterfs

An [Ansible](https://www.ansible.com) role to install/configure [GlusterFS](http://www.gluster.org/)

## Requirements

Add hard drive devices to server nodes to configure for LVM bricks and
set `glusterfs_config_lvm: true`
